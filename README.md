# Aula 3 - HTML: Elementos de Bloco, Linha e Semântica

Este repositório contém os materiais e exercícios práticos da Aula 3 HTML. O objetivo é dominar a diferença entre elementos de bloco e linha, além de implementar estruturas semânticas avançadas em HTML5.

## 📂 Conteúdo do Repositório

### 📄 Documentos HTML (Práticas)
* **`html3_dart.html`**: Exercício prático focado em elementos **inline** específicos para documentação técnica de programação (Linguagem Dart). Utilizar tags como `<code>`, `<var>`, `<kbd>`, `<samp>`, `<sub>` e `<sup>`.
* **`planta_arroz.html`**: Atividade voltada para o ajuste de listas (`li`) como elementos de linha, redimensionamento de imagens e uso da propriedade `float: right` para fluxo de texto ao redor de figuras.
* **`supermercado.html`**: Projeto de estruturação de uma vitrine virtual. Aplicar tags estruturais como `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>` e `<footer>`.

### 📝 Arquivos de Texto (Base para Exercícios)
* **`feijao.txt`**: Conteúdo bruto para a criação da página `feijao.html`.
* **`trigo.txt`**: Texto base para exercícios iniciais de inclusão de imagens no texto e estilos em parágrafos.

### 🖼️ Recursos Visuais
* **`arroz_prato.jpg`**: Imagem ilustrativa para o produto Arroz.
* **`plantando_feijao.jpg`**: Imagem obrigatória para o cabeçalho (`<header>`) do exercício sobre o feijão.
